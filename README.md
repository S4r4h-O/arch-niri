# Arch-Niri

My minimal Arch installation with the [Niri WM](https://github.com/niri-wm/niri).

Easy pre-install with archinstall:

```bash
archinstall --config-url https://github.com/S4r4h-O/arch-niri/raw/refs/heads/main/user_configuration.json
```
